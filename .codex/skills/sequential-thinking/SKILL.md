# Sequential Thinking

Use this skill when a problem is too large to solve in one pass.

## Workflow
1. State the goal.
2. Break it into small steps.
3. Decide the next best action.
4. Re-evaluate after each result.
5. Stop when the outcome is verified.

## Output
- Stepwise reasoning outline
- Decision points
- Next action
