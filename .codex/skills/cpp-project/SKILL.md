# C++ Project

Use this skill when creating or changing a C++ codebase.

## Workflow
1. Identify the build system and compiler targets.
2. Prefer modern CMake and clear project layout.
3. Enforce RAII, smart pointers, and explicit ownership.
4. Add tests and build verification.
5. Keep platform-specific code isolated.

## Output
- Project layout guidance
- Build and test commands
- C++ implementation notes
