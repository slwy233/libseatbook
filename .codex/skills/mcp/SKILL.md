# MCP

Use this skill when connecting tools through MCP.

## Workflow
1. Identify the tool or server you need.
2. Check how it will be launched and authenticated.
3. Keep the server scope narrow.
4. Verify the client can discover the tools.
5. Document the setup and runtime assumptions.

## Output
- Server list
- Connection config
- Verification steps
