# Browser

Use this skill when a task needs UI inspection or browser-driven validation.

## Workflow
1. Open the target page.
2. Reproduce the issue or verify the flow.
3. Inspect visible behavior and console errors if relevant.
4. Capture screenshots when helpful.
5. Confirm the final state.

## Output
- Steps taken in the browser
- UI findings
- Verification result
