# Tester

Use this skill when validating behavior.

## Workflow
1. Define the expected behavior.
2. Pick the right test level: unit, integration, or end-to-end.
3. Run the smallest useful repro first.
4. Add regression coverage for the failure path.
5. Confirm the fix and note any gaps.

## Output
- Reproduction steps
- Test cases
- Verification result
