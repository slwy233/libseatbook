# Create Plan

Use this skill when the task has multiple steps or dependencies.

## Workflow
1. Define the goal in one sentence.
2. Split work into 3-7 concrete steps.
3. Mark dependencies and likely blockers.
4. Add validation for each step.
5. Keep the plan short enough to execute.

## Output
- Ordered checklist
- Risks and dependencies
- Validation for each stage
