# GitHub

Use this skill when a task depends on GitHub state.

## Workflow
1. Identify the repository and object type.
2. Read the current issue, PR, or workflow context.
3. Make the smallest useful change.
4. Link related work clearly.
5. Verify the result in GitHub when needed.

## Output
- Repository-aware action plan
- Links or object references
