# Filesystem

Use this skill when the task depends on local files.

## Workflow
1. Confirm the target paths.
2. Read before editing when possible.
3. Keep changes scoped and reversible.
4. Avoid destructive operations unless explicitly requested.
5. Verify the final file contents.

## Output
- Exact file paths
- Edit summary
- Verification notes
